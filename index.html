Gagasan Kreatif Pemecahan Isu

Isu yang Diangkat
  • ✗ LKH masih menggunakan Microsoft Excel secara manual
  • ✗ Format tidak seragam antar pegawai
  • ✗ Belum ada sistem terintegrasi
  • ✗ Pengisian sering ditunda, dilakukan sekaligus bulanan/tahunan
  • ✗ Belum ada mekanisme monitoring efektif
  • ✗ Dokumen tidak terdigitalisasi, akuntabilitas belum optimal
