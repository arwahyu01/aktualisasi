Latar Belakang|Core Issue

Latar Belakang & Core Issue

Kondisi Sebelumnya
  • ✗ LKH masih menggunakan Microsoft Excel secara manual
  • ✗ Format pelaporan belum seragam antar ASN
  • ✗ Data laporan tidak terintegrasi
  • ✗ Monitoring pimpinan belum optimal
  • ✗ Laporan masih bersifat paper-based
  • ✗ Pengumpulan laporan sering terlambat

Kegiatan 3 / 6|Pembangunan Sistem⭐ Prioritas

Perancangan dan Pembangunan Sistem E-LKH

Tahapan Pembangunan
  • ✓ Pembuatan flowchart sistem
  • ✓ Pembuatan use case diagram
  • ✓ Penyusunan relationship diagram
  • ✓ Penyusunan database (Spreadsheet)
  • ✓ Implementasi Google Apps Script
🔐
Login
Autentikasi ASN dengan email dan password terenkripsi, session token, dan proteksi akses.
📝
Input
Formulir pelaporan harian dengan validasi otomatis, upload dokumen pendukung, dan draft otomatis.
📊
Monitoring
Dashboard real-time untuk pimpinan melihat progres pelaporan ASN, rekap data, dan statistik kinerja.
✅
Approval
Verifikasi dan persetujuan laporan oleh atasan langsung dengan catatan dan status approval.
👤
Akses
Manajemen hak akses berbasis role: Admin, Pimpinan, dan ASN dengan pembatasan fitur sesuai level.
📧
Notif
Notifikasi email otomatis untuk pengingat pengisian LKH, status approval, dan laporan tertunda.
📱
QR Code
Generate QR Code unik per laporan untuk verifikasi cepat dan akses mobile ke detail LKH.
🔄
Reset PW
Fitur reset password mandiri melalui verifikasi email, tanpa harus melalui admin.
📋
Logs
Riwayat aktivitas pengguna tercatat lengkap untuk audit trail dan keamanan data sistem.
